F. Stardew Valley
time limit per test
0.5 seconds
memory limit per test
256 megabytes
input
standard input
output
standard output

Stardew Valley is a single game with liver. Here, you can make friends with NPCs and increase their favorable impression by giving gifts. No matter the player's gender, marriage with the same sex or opposite sex is supported.

During this game, you don't want to marry any NPCs, but you want to keep a good relationship with them. These NPCs are very picky, and only when they are given their favorite gift will they increase their favor. Now there are n NPCs, each of which has its own favorite gift type ai. One NPC can only accept k gifts. When an NPC is given i gift that he likes, his favorable impression will increase by vi. You have nk gifts, each of them has a type gi. Find out the maximum total favorable impression you can get from all NPCs.

Input

The first line of input contains two integers n and (1500,110)k(1n500,1k10) the number of NPCs and the number of gifts each NPC will get.

The second line contains kn integers 1,2,,(1105)g1,g2,,gnk(1gi105) the type of every gift.

The third line contains n integers 1,2,,(1105)a1,a2,,an(1ai105) the favorite gift's type of the NPCs.

The fourth line contains k integers 1,2,,(1105)v1,v2,,vk(1vi105), which is the favorable impression can get from the NPC who gets i frvorite gifts. It is guaranteed that the condition +1vivi+1.

Output

Print one integer the maximum total favorable impression.
